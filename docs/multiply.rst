   {{MacroFunction
   |name=multiply
   |description=
   Returns the numbers that are passed in multiplied together.

   |usage=
   <source lang="mtmacro" line>
   [h: val = multiply(num, num, ...)]
   </source>

   |examples=
   <source lang="mtmacro" line>
       [r: multiply(2, 5, 6)]   
       [r: multiply(4, 3, 2)]
   </source>
   Returns 
       30
       24
   }}
