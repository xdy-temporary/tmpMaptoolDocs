         .. rubric:: floor() Function
            :name: floor-function

         .. container:: template_description

            Returns the number padded in if it is an integer, otherwise
            the number is rounded down to the nearest smaller integer
            and that value is returned.

         .. rubric:: Usage
            :name: usage
