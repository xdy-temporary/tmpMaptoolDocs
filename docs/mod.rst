         .. rubric:: mod() Function
            :name: mod-function

         .. container:: template_description

            Returns the result of the modulo operation between the two
            numbers, which represents the remainder after a division
            operation.

         .. rubric:: Usage
            :name: usage

         .. container:: mw-geshi mw-code mw-content-ltr

            .. container:: mtmacro source-mtmacro

               #. .. code-block:: none

                     [h: val = mod(dividend,divisor)]
