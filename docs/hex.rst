   {{MacroFunction
   |name=hex
   |description=
   Returns a string which contains the hexadecimal representation of the number that was passed in.

   |usage=
   <source lang="mtmacro" line>
   [h: hexVal = hex(num)
   </source>

   |examples=
   <source lang="mtmacro" line>
   [r: hex(23)]
   </source>
   Returns 0x17.

   <source lang="mtmacro" line>
   [r: hex(10)]
   </source>
   Returns 0xA.
   }}
