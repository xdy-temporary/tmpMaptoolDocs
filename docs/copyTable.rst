         .. container:: template_description

            Copies a MapTool table to a new name.

         .. rubric:: Usage
            :name: usage

         .. container:: mw-geshi mw-code mw-content-ltr

            .. container:: mtmacro source-mtmacro

               #. .. code-block:: none

                     copyTable(old name, newname)

         **Parameter**

         -  ``oldname`` - The name of the existing table. It's an error
            if the table doesn't exist.
         -  ``newname`` - The name of the new table to be created. It's
            an error if the table already exists.
